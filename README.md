# CureBuddy
Online Doctor Appointment System
CureBuddy is a full-stack web application designed to simplify the process of booking appointments with doctors. It provides users with a seamless interface to find, book, and manage appointments with healthcare professionals. The platform includes role-based access for patients, doctors, and administrators to efficiently manage appointments, profiles, and availability.

✨ Features
User authentication and role-based access

Doctor profile management

Appointment booking and scheduling

Admin panel for system control

Responsive UI for a smooth experience
